# Sales Performance Dashboard

An interactive **Sales Performance Dashboard** built using **Microsoft Power BI** to analyze sales data and generate actionable business insights.

---

## Overview

This dashboard provides a comprehensive view of sales performance, helping users understand trends, compare product performance, and make data-driven decisions.

---

## Key Features

* **Dashboard Overview** (Provides time-based insights using line charts)
* **Sub-Category Analysis** (Shows Sales, Profit & Quantity trends)
* **Region Analysis** (Visualizes regional performance with stacked & clustered charts)
* **Sales Distribution** (Displays category share using donut charts)
* **Interactive Filters (Slicers)** (Allows dynamic exploration and personalized insights)
* Easy-to-understand and visually appealing layout

3. Explore the dashboard using filters and visuals
